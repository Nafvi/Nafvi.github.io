404 - Page Not Found

Error Description: The requested page does not exist.
Reason 1: Insufficient document access permissions.
Solution:
Change file permissions to 755, for Windows system change directory permissions to writable and readable.
Reason 2: Due to the firewall.
Solution:
Temporarily disable the firewall to allow WWW service.
Reason 3: No default access file in the site root directory.
Solution:
Create an index.html or index.php in the root directory.
Reason 4: Incorrect site configuration directory.
Solution:
Copy the web application to the site directory, or modify the site configuration directory to point to the application directory.
Reason 5: The site uses URL rewriting.
Solution:
Delete the URL rewriting rules, rewrite correct rules, or disable URL rewriting configuration.
