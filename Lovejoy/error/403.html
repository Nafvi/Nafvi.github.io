403 - Forbidden: Access Denied

Error Description: Forbidden access, server denies access.
Reason 1: Default index file not found.
Solution:
In IIS, modify the default document in the 【Enable Default Content Document】 option to the program's home page format, such as: index.html or index.php.
Reason 2: Caused by folder security permissions.
Solution:
Program file - right-click - properties - security - Users - modify to read and execute permissions.
