HTTP 400 - Bad Request

Error Description: Server cannot understand the request due to incorrect syntax.
Reason 1: The client's request does not meet certain restrictions set by the server, or the request itself contains errors.
Solution:
There might be special characters in the link or the link length is too long, please modify accordingly.
Reason 2: Caused by too large request header or cookie
Solution:
Use the ctrl+shift+delete shortcut to clear cookies.
