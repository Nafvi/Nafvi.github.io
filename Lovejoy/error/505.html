HTTP 505 - HTTP Version Not Supported

Error Description: HTTP version not supported.
Reason 1: Your Web server does not support, or refuses to support the HTTP protocol version specified in the HTTP request data stream sent by the client (such as your browser).
Solution:
Upgrade your Web server software.
Reason 2: Error in the HTTP request format.
Solution:
Review your code. The problem might be an extra space added after the request. The HTTP protocol is very strict.
